## Prompts Úteis para Engenharia de Dados

```
# Explicar conceito
"Explique [conceito] como se eu fosse um iniciante. Use analogias práticas."

# Gerar exercício
"Crie um exercício prático de [tema] com solução comentada."

# Revisar código
"Revise esse código Python/SQL e aponte melhorias de performance e legibilidade."

# Gerar pipeline
"Crie um pipeline ETL em Python que lê de [fonte], transforma [regra] e carrega em [destino]."

# Tirar dúvida
"Estou estudando [módulo]. Não entendi [conceito]. Me explica com exemplo real."
```
