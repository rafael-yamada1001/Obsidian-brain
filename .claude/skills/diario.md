Crie uma nota de diário para hoje no vault do usuário.

1. Use a data de hoje no formato AAAA-MM-DD
2. Crie o arquivo em `Diario/AAAA-MM-DD.md` (com a data real)
3. Use este template:

```
# AAAA-MM-DD

## 🎯 Foco do dia
<!-- O que é mais importante hoje? -->

## 📚 Estudos
<!-- O que estudei / vou estudar hoje? -->
- Curso: 
- Tópico: 

## 💡 Ideias e Pensamentos
<!-- Qualquer coisa que surgiu -->

## ✅ Conquistas
<!-- O que completei hoje -->

## 🔗 Links do dia
<!-- Notas relacionadas -->
```

4. Informe o usuário que a nota foi criada e está pronta para preenchimento.
