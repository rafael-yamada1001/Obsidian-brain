Resuma a conversa atual em formato compacto e salve no vault se o usuário quiser.

1. Analise toda a conversa atual
2. Gere um resumo com este formato:

```
## TL;DR — AAAA-MM-DD HH:MM

### O que foi feito
- 

### Decisões tomadas
- 

### Próximos passos
- 

### Arquivos alterados no vault
- 
```

3. Mostre o resumo ao usuário
4. Pergunte: "Quer salvar no vault?" 
   - Se sim: salve em `Diario/AAAA-MM-DD.md` (crie se não existir) na seção correspondente
   - Se não: apenas exiba
